# EMC Academy 🚀

Responsive registration website for EMC Academy - Training | Tech News | Freshers Guide.

Academic project built with HTML5 + CSS3 to demonstrate modern responsive web design and form handling.

### ✨ Features
- Responsive design for desktop, tablet, and mobile
- Home page with Full Stack course breakdown: Frontend, Backend, Database
- Student registration form: name, age, gender, email, course selection, resume upload
- Form validation: required fields, email format, PDF file restriction
- Embedded YouTube intro video for EMC Academy
- Custom favicon and logo branding
- Clean UI with card layout, shadows, and hover effects
- Built with pure HTML/CSS - no frameworks

### 🛠️ Tech Stack
- HTML5
- CSS3
- VS Code + Live Server
